Event Images Batch Processor

How to Use

  • Upload a ZIP file containing event images (JPEG/PNG format).
  • Upload a transparent university logo (PNG format).
  • Upload a transparent chapter logo (PNG format).
  • Click the Process Images button to generate a new ZIP file with the logos applied.
  • Share the downloaded event images! Made with ❤️ by Shahzeb Ali.
